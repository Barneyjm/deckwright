The problem

Slides are everywhere. Control over them isn't.

Set up the pain before the solution.
Why this matters

The deck is fine. The plumbing isn't.

Decks drift across versions, fonts substitute on the wrong machine, files balloon, and the format is locked to one vendor's app.

  • Fonts break the moment the file leaves your laptop.
  • Layout shifts on a different screen or projector.
  • Live data means re-exporting, every single time.
  • Distribution requires the recipient to own the app.

The approach

Lock the canvas. Embed everything.

What "indistinguishable" requires

Three guarantees PowerPoint gives for free.

16:9
A fixed canvas that scales to any display instead of reflowing like a web page.
100%
Fonts embedded in the file, so type never substitutes on another machine.
0deps
No external requests — it renders identically offline, forever.
These three are the entire trick to passing the executive sniff test.

How you'd use it

Author in text. Present like a pro.

For the author

Each slide is one <section class="slide">. Pick a layout, drop in your words. Add class="frag" to anything you want to reveal on click.

For the room

Arrow keys or a clicker. Tap and swipe on a tablet. F for fullscreen, S for speaker notes, Cmd + P to hand out a clean PDF.

Demonstrate: press S now to show this very note in the speaker panel.
